## KPI gate

Stop scaling outreach volume when any of these signal "we found a channel that works":

- **B2B:** ≥ 5% DM reply rate AND ≥ 10% reply-to-demo-booked AND $500+/mo closed
- **Creator:** ≥ 1 creator driving >100 free signups in 2 weeks
- **SEO/directory:** ≥ 1 directory listing driving >50 clicks/week

If none of these hit at day 30, the product is the bottleneck (not acquisition). Re-evaluate before scaling further.
