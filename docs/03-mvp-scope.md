# MVP Scope — Calora

## Must-have (ship blocker)
- [ ] Camera capture (front/back) + photo library upload
- [ ] Image upload to API → returns calorie/macro estimate
- [ ] Edit screen (user can adjust portion, fix items)
- [ ] Save to daily log
- [ ] Today's total + ring progress vs goal
- [ ] Text input fallback ("2 eggs, toast with butter")
- [ ] History (last 7 days)
- [ ] Goal setting (manual kcal target)

## Nice-to-have (post-MVP)
- [ ] Barcode scanner
- [ ] Exercise tracking
- [ ] Weekly/monthly trends chart
- [ ] Export to CSV
- [ ] Multi-user auth + cloud sync
- [ ] Meal templates ("my usual breakfast")
- [ ] Recipe calorie calculator

## Explicitly NOT in MVP
- ❌ Social features
- ❌ AI coach / chat
- ❌ Wearable integration
- ❌ Micronutrient tracking
- ❌ Restaurant menu database

## Acceptance test (MVP done = )
A new user can:
1. Open the app on their phone
2. Snap a photo of lunch
3. See "chicken rice + salad, ~520 kcal" within 10 seconds
4. Tap save
5. See today's total update with a ring filling up

…all in under 30 seconds, no signup.
